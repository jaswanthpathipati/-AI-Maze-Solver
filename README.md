## 📌 Overview

The **AI Maze Solver** is a visually interactive web-based application that allows users to **create, customize, and solve mazes** using intelligent search algorithms.

It demonstrates core concepts of **Artificial Intelligence, graph traversal, and pathfinding** through a modern UI with real-time animations.

---

## 🚀 Features

### 🎮 Interactive UI

* Neon-themed modern interface
* Real-time maze visualization
* Smooth pathfinding animations

### 🧱 Build Mode

* Draw walls manually
* Erase paths
* Set **Start** and **End** points dynamically

### 🤖 Algorithms Supported

* **BFS (Breadth-First Search)** → Guarantees shortest path
* **DFS (Depth-First Search)** → Deep exploration
* **A* (A-Star)** → Fastest and most efficient

### 🎯 Maze Generation

* Generate random mazes with:

  * Easy
  * Medium
  * Hard difficulty levels

### ⚡ Speed Control

* Adjust algorithm speed:

  * Fast
  * Medium
  * Slow

### ⚙️ Additional Options

* Toggle exploration visualization
* Clear/reset maze instantly

---

## 🧠 Algorithms Explained

### 🔹 Breadth-First Search (BFS)

* Explores nodes level by level
* Always finds the shortest path

### 🔹 Depth-First Search (DFS)

* Explores as deep as possible
* May not find the shortest path

### 🔹 A* Algorithm

* Uses heuristics (distance estimation)
* Most optimized and widely used
